# Disease-Prediction

-The developed system successfully predicts possible diseases based on the symptoms entered by the user using Machine Learning algorithms. After training the model on the dataset containing multiple symptoms and diseases, the system is able to analyze user inputs and provide accurate disease predictions.
-The application also displays precautionary measures related to the predicted disease, helping users understand how to manage or reduce the risk of the illness. This makes the system not only a prediction tool but also a guide for basic health awareness.
